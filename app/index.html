🛢️ DWCMission - Tank Management

SAP CAP + SAP Datasphere Integration
Bienvenue dans l'application DWCMission.
Cette application CAP expose un service OData v4 pour la gestion des réservoirs.
Les données master sont lues depuis SAP Datasphere. Les lectures manuelles sont stockées dans SAP HANA Cloud.

📡 Endpoints OData v4

Service Metadata

Accédez aux métadonnées complètes du service OData v4 (définition des entités et relations).

Voir $metadata

Tanks (Réservoirs)

Liste des réservoirs chargés depuis SAP Datasphere. Données en lecture seule.

Voir les Tanks

TankVolumes (Lectures)

Lectures manuelles de volume saisies par les techniciens terrain. CRUD complet.

Voir les TankVolumes
